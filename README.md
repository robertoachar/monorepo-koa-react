# @monorepo/koa-react

[![License][license-badge]][license-url]

> Monorepo for Koa and React.

# Development

- Clone the repo

```bash
$ git clone https://github.com/robertoachar/@monorepo/koa-react.git
```

- Install dependencies

```bash
$ npm install
```

- Run scripts

| Action                 | Usage          |
| ---------------------- | -------------- |
| Start development mode | `npm start`    |
| Lint code              | `npm run lint` |
