# @monorepo/server

> Server.

# Development

- Clone the repo

```bash
$ git clone https://github.com/robertoachar/@monorepo/server.git
```

- Install dependencies

```bash
$ npm install
```

- Run scripts

| Action                 | Usage          |
| ---------------------- | -------------- |
| Start development mode | `npm start`    |
| Lint code              | `npm run lint` |
